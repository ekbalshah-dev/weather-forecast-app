# Weather Forecast App

A web application that displays real-time weather information for any city using a weather API.

## Features
- Search weather by city name
- Displays temperature and weather conditions
- Fetches live data from a weather API
- Dynamic updates using JavaScript

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

## How It Works
The application sends a request to a weather API when a user enters a city name.  
The API returns weather data in JSON format, which is processed using JavaScript and displayed on the webpage.

## How to Run
1. Clone the repository
2. Open the project folder
3. Open `index.html` in a web browser
